# [Maven Helper](https://plugins.jetbrains.com/plugin/7179-maven-helper)

Maven Helper 可以帮助我们在 IDEA 中更好地开发 maven 项目，主要功能包括：

- 更方便地执行 maven 命令；
- 分析和排除 Maven 冲突；
- 自定义 Maven goals；
- 在当前 Module 所在的路径中打开 Terminal；
