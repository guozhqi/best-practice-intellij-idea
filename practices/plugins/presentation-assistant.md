# [Presentation Assistant](https://plugins.jetbrains.com/plugin/7345-presentation-assistant)

Presentation Assistant 是一款快捷键展示插件，适合在直播或者录屏的场景使用。
如果你触发了某一个操作（不管是不是使用快捷键触发的），这款插件都会在屏幕上展示你可以使用哪些快捷键触发同一操作，方便他人复现操作步骤。
