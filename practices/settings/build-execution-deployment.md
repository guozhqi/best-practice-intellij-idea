# Build,Execution,Deployment

## Build Tools

### Maven

IDEA 的 Maven 配置建议作以下设置：

- "Maven home path": 建议设置为自己本机安装的 maven 路径；
- "User settings file" 和 "Local repository" 也建议使用和本机相同的配置；

这样可以使 maven 的行为在 IDEA 和 命令行中保持一致。
